Query Save

Type the name of the Query and click either the Cancel button to Cancel the save, or the Save button to save the Query. If a query is being saved as the same name as an existing query the dialog will not close. To add replace an existing query click the (toggle) button next to Save and choose the Save As button.

Saved queries appear in the Sidebar and can be reused and configured by choosing Tabs > Configure Sidebar (when a query tab is active) and choosing Configure Queries.
