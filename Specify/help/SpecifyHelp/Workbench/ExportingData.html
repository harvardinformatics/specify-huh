Exporting Data

The WorkBench can export an entire Data Set or select records within a Data Set by choosing action items from either the Sidebar or the Workspace. The following table describes each of these export functions:

 

Button
Item
Item Location
Explanation
Export Data Set
side bar

Exports an entire Data Set.

Only closed Data Sets can be exported by this method.

Export Excel Template
side bar

Exports a template for Data Sets to Excel.

Only closed Data Sets can be exported by this method.

Export Data Set
works pace

Exports select records or the entire Data Set (by selecting all records).

Only active Data Sets can be exported by this method.
