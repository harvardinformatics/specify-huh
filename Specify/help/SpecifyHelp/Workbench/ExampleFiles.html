Examples

Example files and images are included with the WorkBench and can be imported for testing the various functions of the WorkBench. These are located in the Examples/Files directory. Each file includes several columns representing different discipline specific fields. Each file also includes columns that do not map in the WorkBench Data Set Mapping Editor to illustrate the operation of this function. The resulting Data Sets from these files will exhibit different functionality in the BioGeomancer() and Google Earth () tools. The features of each file are listed below by discipline. The Ichthyology and Botany files have been imported into the WorkBench at startup.

Ornithology MS Excel file - 41 records of various species from China:

  • Tissue number and Collector number fields do not map.
  • BioGeomancer - the tool is inactive because the county field is missing which is evident in the tooltip for this tool.
  • Google Earth displays the records and associated data.

Botany MS Excel file - 74 records of various species collected in Douglas County, Kansas:

  • Phenology and General Remarks fields do not map.
  • BioGeomancer returns 51 results for the given data set.
  • Google Earth displays the records and associated data.

Entomology MS Excel file - 39 records of various species collected in Kansas and Peru:

  • All fields automap.
  • BioGeomancer returns 1 result for the given data set.
  • Google Earth displays the records and associated data.

Ichthyology MS Excel file - 201 records of various tissues collected in the Seychelles:

  • File or URL link and Voucher # fields do not map.
  • BioGeomancer returns 116 results for the given data set.
  • Google Earth displays the records and associated data together with specimen images. The File or URL link field has been formatted with HTML code to allow linking of the images into each pushpin balloon in Google Earth. These can be clicked on to view full size in a browser window.

Herpetology MS Excel file - 30 records of various species collected in Dominican Republic, Haiti, El Salvador and the Philippines:

  • All fields automap.
  • BioGeomancer returns 8 results for the given Data Set.
  • Google Earth displays the records and associated data.

Mammalogy MS Excel file - 43 records of various species collected in the state of Maine:

  • Collector number field does not map.
  • Biogeomancer returns 42 results for the given Data Set.
  • Google Earth displays the records and associated data.

 

Example data entry images have also been provided for Ornithology, Botany, Ichthyology and Mammalogy data. These images are of labels taken from the University of Kansas Biodiversity Research Center collections and can be imported using the Import Images action on the sidebar for testing data entry from labels. These are located in the Examples/Images directory.
