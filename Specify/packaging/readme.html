Release Notes Specify 6.3.05

01 September 2011

List of changes in Specify 6.3.05
  1. Two users can now simultaneously edit the same user defined picklist without causing an error. 2011-09-01 (Bugzilla 8527)
  2. Opening a WorkBench data set with associated images no longer generates an error. 2011-09-01 (8525)
  3. An issue causing improper validation of required fields in the WorkBench uploader has been fixed. 2011-09-01 (8523)
  4. "Preparation Description 2" is no longer mapped to "Preparation Description 1" in the WorkBench Uploader. 2011-09-01 (8522)
  5. The Import File Splitter no longer changes the format of dates from the Excel file when splitting it. 2011-09-01 (8509)
