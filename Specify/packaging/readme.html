Specify 6.1
30 October 2009
Release Notes
Specify Software Project


New Capabilities:

  1. Added support for automatic generation of Life Science Identifiers (LSIDs) for Agents, Chronostratigraphy, Collection Objects, Geography, Journals, Lithostratigraphy, Localities, Reference Works, and Taxonomy.
  2. Added NASA World Wind plug-in for geo-referencing specimen localities.
  3. Added support for the GBIF IPT toolkit for publishing XML structured data with the TAPIR protocol to the GBIF cache and to other specimen database portals.
  4. Added database audit records which identify changed records.
  5. Added support for polygon boundaries for localities and added fields to the GeoCoordinate Detail Table to accommodate ever shifting MANIS Network requirements for mammal collections.
  6. Added several requested fields to Specify WorkBench schema
  7. Added capability to extract Day, Month, Year from date fields in queries and reports.

A new Specify Mobile WorkBench application and a new easy-install version of Specify 6, Specify EZDB will be released shortly as an update Specify 6.1.1.
