Specify 6.0 Release Notes (readme.html):

 

  • Specify 6.0 is a wholly new implementation of Specify written in Java as a cross-platform application for managing biological collections data.
  • Released 10 April 2009
  • Licensed under GNU GPL2.
  • Installation packages are available for Windows, MacOS X 10.5+ and Linux. To run Specify 6, JRE 6.0+ and the MySQL 5.1 data manager are required, see the documentation files on the Specify web site for instructions on how to install Java and MySQL prior to installing Specify.
  • The Specify wizard (SpWizard.exe) installed in the Specify /bin directory is used for creating an initial biological collection in a Specify installation. See the embedded help file in the Wizard for assistance in configuring the first Specify collection.
  • Additional installation documentation will be available soon on the Specify Software web site (specifysoftware.org).

Description of Changes:

The Specify Web site will have complete documentation of the changes to Specify and a description of its many features soon.

See the embedded Specify help file, which will be available online on the Specify web site.

For additional information, please visit the project website at www.specifysoftware.org
