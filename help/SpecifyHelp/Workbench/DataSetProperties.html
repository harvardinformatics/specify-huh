Data Set Properties

Data Sets are named and described as part of the Data Set Properties.

To view and/or edit the properties of a Data Set, right click on a Data Set name on the side bar. A context menu will appear:

 

Click 'Edit Properties' to display the Data Set Properties dialog:

Data Set Properties

 

If a Data Set was created from an imported file, the path to the file will display in the 'Source File' area of the dialog.
