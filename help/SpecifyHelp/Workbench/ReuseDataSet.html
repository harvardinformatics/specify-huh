Reuse Data Set

The WorkBench can create a new Data Set from the structure of any existing Data Set. This eliminates the need to choose the Columns for the new Data Set.

 

Drag and Drop

The fastest way to create the new Data Set from an existing structure is to 'drag and drop' the Data Set name onto the 'New Data Set' action on the side bar. This eliminates the need to choose an existing Data Set to reuse.

OR, click on the 'New Data Set' item on the side bar. Choose "Use an existing Data Set mapping".

Choose a Data Set to copy the structure from and click 'ok'. The Data Set Properties dialog will then appear to name and describe the new Data Set:

The new Data Set will appear in the work space.
